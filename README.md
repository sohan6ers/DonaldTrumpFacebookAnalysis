# Donald Trump Facebook Analysis

## About Dataset
Dataset has been extracted from Facebook. As dataset is quite large, this project utilises shell commands in terminal to process and analyse the large dataset.
The columns include:
- page name
- identifiers
- post name
- message
- description
- caption
- post type
- likes count
- comments count
- shares count
- reaction count
- post link
- picture
- post time
